Angular utiliza el Router para navegar entre componentes. Define las rutas en un array de Routes y usa RouterModule.forRoot(routes). 

const routes: Routes = [
  { path: '', component: HomeComponent },
  { path: 'instalacion', component: InstalacionComponent },
  { path: 'estructura-proyecto', component: EstructuraProyectoComponent },
  ...
];

Agrega directivas como [routerLink] en tu HTML para crear enlaces entre rutas.
